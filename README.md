# Data Plotting Program

## Description
This program reads data from a CSV file and creates a scatter plot using Matplotlib.  
The user specifies which columns to plot on the x and y axes.

---

## Features
- Reads CSV data files  
- Selects columns by name  
- Creates a labeled scatter plot  

---

## Requirements
- Python  
- Matplotlib

Install Matplotlib with:
```bash
pip install matplotlib
```
